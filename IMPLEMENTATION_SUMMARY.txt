===============================================================================
TROUBLESHOOTING
===============================================================================

Problem: Logo not showing
Solution:
  1. Check file exists: ls design_app/public/images/cnit-logo*.svg
  2. Clear cache: bench --site your-site clear-cache
  3. Rebuild: bench build --app design_app
  4. Restart: bench restart
  5. Hard refresh browser: Ctrl+Shift+R

Problem: Colors not applied
Solution:
  1. Hard refresh browser: Ctrl+Shift+R
  2. Check console for errors: F12
  3. Rebuild: bench build --app design_app
  4. Verify app installed: bench --site your-site list-apps

Problem: Styles look broken
Solution:
  1. Check hooks.py includes CSS files
  2. Verify file names match exactly
  3. Rebuild: bench build --app design_app
  4. Restart: bench restart
  5. Check browser DevTools for 404 errors

Problem: Footer not visible
Solution:
  1. Scroll to bottom of page
  2. Check cnit-branding.css is loaded (DevTools Network tab)
  3. Verify .layout-footer element exists on page
  4. Check CSS specificity conflicts
